# 2차원 배열과 좌표 개념 정리 (백준 2563 색종이 기준)

## 1. 2차원 배열 기본 구조

파이썬에서 2차원 배열은 다음과 같은 형태로 구성된다.

``` python
board = [[0]*100 for _ in range(100)]
```

접근 방식은 항상 다음과 같다.

board\[행\]\[열\] board\[i\]\[j\]

-   i → 행 (세로, 줄)
-   j → 열 (가로, 칸)

------------------------------------------------------------------------

## 2. 입력 좌표 (x, y)와 배열 인덱스의 관계

색종이 문제에서 입력은 다음과 같다.

x, y

-   x → 가로 방향 (열)
-   y → 세로 방향 (행)

하지만 배열 접근은 다음과 같이 해야 한다.

board\[y\]\[x\]

즉, 수학 좌표 (x, y)와 배열 인덱스 \[y\]\[x\]는 뒤집힌 구조이다.

------------------------------------------------------------------------

## 3. 색종이 문제 핵심 로직

1.  100x100 배열 생성 (초기값 0)
2.  색종이 개수 입력
3.  (x, y) 기준으로 10x10 영역을 1로 표시
4.  전체 배열을 순회하며 1의 개수 카운트

구현 예시:

``` python
import sys
input = sys.stdin.readline

blank = [[0]*100 for _ in range(100)]
n = int(input())

for _ in range(n):
    x, y = map(int, input().split())
    for i in range(y, y+10):      # 행
        for j in range(x, x+10):  # 열
            blank[i][j] = 1

count = 0
for i in range(100):
    for j in range(100):
        if blank[i][j] == 1:
            count += 1

print(count)
```

------------------------------------------------------------------------

## 4. 행/열 헷갈리지 않는 방법

기억 공식:

-   i는 세로 (위→아래)
-   j는 가로 (왼→오)

또는

board\[세로\]\[가로\] board\[y\]\[x\]

------------------------------------------------------------------------

## 5. 오늘 느낀 점

-   좌표 (x, y)와 배열 인덱스 \[y\]\[x\]는 항상 뒤집힌다.
-   2차원 배열 문제의 핵심은 좌표 변환 이해이다.
-   중복 처리는 if문 없이 덮어쓰기 방식이 더 깔끔하다.
-   구현 문제는 개념보다 좌표 체계 이해가 더 중요하다.
